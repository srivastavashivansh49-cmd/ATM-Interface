## 📌 About the Project

**ATM Interface** is a simple web-based project that simulates basic ATM operations.

Users can check their balance, deposit money, withdraw money, and exit the ATM interface. The project is made using **HTML**, **CSS**, and **JavaScript**, making it beginner-friendly and useful for practicing JavaScript logic and DOM manipulation.

---

## ✨ Features

- 💰 Check account balance
- ➕ Deposit money
- ➖ Withdraw money
- 🚪 Exit message
- 🎨 Simple handmade-style user interface
- 📱 Responsive design
- ⚡ Fast and lightweight
- 🧑‍💻 Beginner-friendly code structure
